# Система банковских счетов (Bank Account System)

Автор: Никита Вьюнов

## Описание
Система банковских счетов - это консольное приложение, написанное на Python. Оно симулирует базовые банковские операции, позволяя пользователям создавать различные типы счетов (Текущий, Сберегательный, Кредитный), выполнять транзакции (пополнение, снятие, перевод) и просматривать историю операций.

## Архитектура
Приложение строго следует шаблону проектирования MVC (Model-View-Controller):
- Model (Модель - BankModel, Account, Transaction): Обрабатывает логику данных, состояние, бизнес-правила (например, предотвращение превышения лимита) и сериализацию/десериализацию JSON.
- View (Представление - BankView): Управляет всем взаимодействием с пользователем, запросами ввода и выводом в консоль.
- Controller (Контроллер - BankController): Выступает связующим звеном, принимая пользовательский ввод от View, управляя Model и возвращая обновленные данные обратно во View.

В системе используются концепции объектно-ориентированного программирования (ООП), включая наследование и полиморфизм, а также структура данных Queue (очередь) для управления историей транзакций.

## Функционал
- Создание Текущих (Checking), Сберегательных (Savings) и Кредитных (Credit) счетов.
- Внесение и снятие средств со строгой проверкой валидности.
- Перевод средств между счетами.
- Фильтрация истории транзакций по дате и типу операции.
- Постоянное хранение данных с использованием формата JSON.

## Как запустить
1. Убедитесь, что на вашем компьютере установлен Python версии 3.x.
2. Клонируйте этот репозиторий или скачайте исходный код.
3. Откройте терминал/командную строку и перейдите в папку проекта.
4. Запустите приложение с помощью следующей команды:
   ```bash
   python main.py
   ```

## Примеры использования
1. Создание счета:
    - Выберите 1 в главном меню.
    - Введите тип 1 для Текущего счета (Checking).
    - Введите ID счета 101.
    - Введите имя владельца Иван Иванов.`

2. Пополнение счета:
    - Выберите 2 в главном меню.
    - Введите ID счета 101.
    - Введите сумму 500.
3. Перевод средств:
    - Убедитесь, что у вас есть второй счет (например, ID 102).
    - Выберите 4 в главном меню.
    - Введите ID счета отправителя 101, ID счета получателя 102 и сумму 100.

4. Фильтрация истории:
    - Выберите 6 в главном меню.
    - Введите ID счета 101.
    - Нажмите Enter, чтобы пропустить фильтр по дате, и введите Перевод для фильтрации по типу.
